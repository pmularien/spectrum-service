spectrum-service
================

Service layer for the spectrum library.

The service utilizes:
- the Play2 framework
** https://www.playframework.com
- Swagger
** http://swagger.io
- dropwizard metrics (formerly codahale metrics, formerly yammer metrics)
** http://dropwizard.io
- Grizzled SLF4J
** http://software.clapper.org/grizzled-slf4j/
